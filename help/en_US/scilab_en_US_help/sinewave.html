sinewave

Return an M-element vector with I-th element given by 'sin(2* pi *(I+D-1)/N).'

Calling Sequence

y= sinewave(M)
y= sinewave(M,N)
y= sinewave(M,N,D)

Parameters

M:

Input vector

N:

The default value for N is M

D:

The default value for D is 0

AMPL:

Amplitude

Description

Return an M-element vector with I-th element given by 'sin(2* pi *(I+D-1)/N).'
