synthesis

Compute a signal from its short-time Fourier transform

Calling Sequence

X= synthesis(Y,C)

Parameters

Y:

Shirt-time fourier transform

C:

3-element vector C specifying window size, increment, window type.

Description

Compute a signal from its short-time Fourier transform Y and a 3-element vector C specifying window size, increment, and window type. The values Y and C can be derived by [Y, C] = stft (X , ...)
