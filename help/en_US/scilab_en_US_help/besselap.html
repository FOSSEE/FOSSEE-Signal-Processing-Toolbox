besselap

Return bessel analog filter prototype.

Calling Sequence

[zero, pole, gain] = besselap (n)
[zero, pole] = besselap (n)
zero = besselap (n)

Parameters

n:

Filter Order

zero:

Zeros

pole:

Poles

gain:

Gain

Description

This is an Octave function. It Return bessel analog filter prototype of nth order.

Examples

[zero, pole, gain] = besselap (5)
zero = [](0x0)
pole =

-0.59058 + 0.90721i
-0.59058 - 0.90721i
-0.92644 + 0.00000i
-0.85155 + 0.44272i
-0.85155 - 0.44272i

gain =  1
